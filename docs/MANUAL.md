# 操作マニュアル

## 起動方法

```bash
uv run python -m app.main
```

---

## 画面説明

### 設定エリア（画面上部）

| 項目 | 説明 |
|---|---|
| **モデルサイズ** | `tiny`（速い・粗い）〜 `large`（遅い・精確）。会議ならまず `base` か `small` を試してください |
| **言語** | 日本語の会議なら「日本語固定」推奨。混合言語なら「自動検出」 |
| **温度** | 0.0 が最も安定した出力。誤認識が多い場合は少し上げてみる |
| **英語に翻訳して出力** | ONにすると文字起こしと同時に英語訳を出力します |
| **初期プロンプト** | 頻出する固有名詞・人名・専門用語をスペース区切りで入れると認識精度が上がります |

> 設定は変更のたびに自動保存されます。次回起動時も維持されます。

---

### 録音機能

1. 赤い丸ボタン（🔴）をクリックすると録音開始
2. ボタンが四角（■）に変わり、「録音中…」のステータスが表示されます
3. もう一度クリックすると録音停止 → WAV ファイルとして自動保存
4. そのまま文字起こしが自動で開始されます

録音ファイルの保存場所: `~/.whisper_flet_transcriber/recordings/record-YYYYmmdd-HHMMSS.wav`

---

### ファイル選択エリア

1. **「音声ファイルを選択」** ボタンをクリック
2. ファイルダイアログから音声ファイル（mp3, wav, m4a など）を選択
3. 選択したファイル名が表示されます

---

### 文字起こし実行

1. **「文字起こし開始」** ボタンをクリック（ファイル選択時のみ有効）
2. プログレスバーが進捗率（0〜100%）を表示しながら変換が進みます
3. 完了すると「✅ 完了」と表示され、結果が下部に表示されます

> モデルサイズによっては数分かかる場合があります。初回実行時はモデルのダウンロードも行われます。

---

### 結果エリア（画面下部）

- **コピーボタン（📋）**: クリップボードに全文コピー
- **保存ボタン（💾）**: テキストファイル（.txt）として任意の場所に保存
- テキストボックス内をクリックして直接選択することも可能

## よくある質問

**Q: 初回起動が遅い**  
A: 選択したモデルをダウンロードしています（数百MB〜数GB）。2回目以降は速くなります。

**Q: 日本語の認識精度が低い**  
A: モデルを `small` 以上に変更する、または初期プロンプトに関連用語を入れてみてください。

**Q: 録音できない / マイクが認識されない**  
A: OSのマイクアクセス許可を確認してください（macOS: システム設定 → プライバシーとセキュリティ → マイク）。

**Q: エラーが出る**  
A: ffmpeg がインストールされているか確認してください（`ffmpeg -version`）。
